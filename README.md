# Encryption-Program
A program to encrypt and  decrypt files with ease and a difficult-to-crack algorithm (hopefully)

Made by me cos my school suffered one or two (actually a few) data breaches and email hacks etc, and so I decided to apply my programming skills to the task.

The idea is to take a message, and encrypt it in a completely unorthodox way which can't really be deciphered without a copy of this program. Apparently this could also be useful for my Maths IA in my IB, so yay!

If you notice the public and private keys in the code, those have nothing to do with RSA encryption. They are more to do with an extra server-level layer of encryption, so that messages intercepted between people on the same server cannot be decrypted without a copy of the maps of public keys.
